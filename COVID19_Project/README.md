This project explores the COVID-19 Patient Recovery Dataset from Kaggle, containing 70,000 anonymized patient records. The analysis includes exploratory data analysis (EDA), visualizations, and predictive insights into recovery, mortality, disease severity, and ICU admissions. Key factors such as age, gender, comorbidities, vaccination status, and treatments are examined to understand patterns and trends in COVID-19 outcomes. Open the `covid19_recovery_analysis.ipynb` notebook to view the full analysis and visualizations.
